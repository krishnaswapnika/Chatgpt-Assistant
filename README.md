
# ChatGPT-4 Assistant

An AI Assistant chat bot created using react


## Tech Stack
- ReactJS
- NodeJS Server
- Web Socket (socket.io)
- OpenAI API (Rapid API)


## Deploy

Live Link: https://chatgpt-assistant.vercel.app


## Installation

Install backend server

```bash
  npm install 
```

Install frontend server

```bash
  cd ./frontend
  npm install 
  npm start
```
